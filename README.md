# BlazeFace

![demo](demo.gif)

MediaPipe BlazeFace for realtime face detection in Unity Engine.

## Requirements
- Unity 2021.2+

## Supported Platforms
- Android API level 24+
- iOS 13+
- macOS 10.15+ (Apple Silicon and Intel)
- Windows 10+ (64-bit only)
- WebGL:
    - Chrome 91+
    - Firefox 90+
